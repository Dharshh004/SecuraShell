# 🔐 SecuraShell Studio

A beginner-to-intermediate Bash-based cybersecurity toolkit built for educational purposes.

## 📌 Features

- ✅ Custom Port Scanner (with logging & scan modes)
- ✅ Subnet Scanner using Nmap
- ✅ Password Strength Checker (with entropy estimation)
- ✅ Dark-themed Web Interface
- ✅ Logging system for scan results
- ✅ IP validation and error handling

---

## 🛠 Technologies Used

- Bash Scripting
- Nmap
- HTML5
- CSS3
- JavaScript
- Kali Linux
